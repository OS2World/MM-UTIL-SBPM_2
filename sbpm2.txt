Hallo Allemaal!

Wat is nieuw

0.99b
- Override MMPM/2 kan nu op 2 snelheden worden ingesteld, nl 300 ms en
  1 seconde.

0.98b
- Override MMPM/2 aangepast om maar 1 seconde te wachten ( op idle priority )
- Communicatie tussen sliders verbeterd ( minder overhead 0
- SBPM/2 nu ook via internet op te halen

0.97b
- Mogelijkheid om geminimaliseerd te starten toegevoegd
- Eerste gedeelte advanced settings geimplementeerd
  - SBPRO, filters en input selection )
  - SB16, speaker volume, Gain In, Automatic Gain

0.95b

- Het is nu mogelijk om MMPM/2 te omzeilen door het gebruik van de
  optie 'Override SBPM2' Als deze is geactiveerd wordt elke 10 seconden
  de ingestelde volumes opnieuw weggeschreven naar de soundblaster.
- Setup procedure is verbeterd, SBPM/2 vraagt nu automatisch naar de settings.
- Programma is kleiner gemaakt.

Mogelijkheden

- SBPro en SB16 compatible
- Op de SBPRO: VOC/FM/Mic/CD/LineIn/Main
- Op de SB16 : VOC/FM/Mic/CD/LineIn/Main/Trebble/Gain/Bass
- Persistant sliders ( SBPM/2 onthoudt de standen van de sliders
- Basispoort instelbaar tussen 210 tot 250 ( via de resource file tot 410
  uitbreidbaar )
- 'Lock Left & Right' slider
- SoundBlaster kan tijdens 't starten van SBPM/2 gereset worden.
- De interface kan via de meegeleverde resources aangepast worden naar
  eigen smaak.

Hoe te gebruiken:

Met de environment variabele SBPM2 kan de naam en plaats van de ini file worden
opgegeven. Bijvoorbeeld set SBPM2=E:\VP\OUT\SBPM2.INI ( default is sbpm2.ini
in de huidige directory )

Daarna het programma starten en instellen of er een SB Pro of SB 16 wordt
gebruikt. Daarna moet e.e.a. duidelijk zijn.
